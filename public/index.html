URL Shortener Microservice API


This API is a Free Code Camp - Back End Development Certification project.

This service creates a shortened url link for a url passed to the api via /new/<url_goes_here>

A JSON object is sent in the response containing the shortened url with property "short_url" along with the original url with property "original_url".

Example Usage

https://url-short-nburazor.herokuapp.com/new/https://www.google.com

https://url-short-nburazor.herokuapp.com/new/https://www.freecodecamp.com

Example Output

{ "original_url":"https://www.freecodecamp.com", "short_url":"https://url-short-nburazor.herokuapp.com/2" }

Following the "short_url" link will redirect to the "original_url"
